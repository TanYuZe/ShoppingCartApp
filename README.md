# Shopping Cart System

This is a full-stack shopping cart system with Spring Boot backend and React TypeScript frontend.

## Backend

- Spring Boot 3.2.0
- PostgreSQL database
- JPA/Hibernate
- REST API

### Running the Backend

1. Ensure PostgreSQL is running and create a database named `shoppingcart`.
2. Update `application.properties` with your database credentials.
3. Run `mvn spring-boot:run` in the backend directory.

## Frontend

- React 18
- TypeScript
- Vite

### Running the Frontend

1. Run `npm install` in the frontend directory.
2. Run `npm run dev` to start the development server.
